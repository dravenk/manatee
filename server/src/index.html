🚀 数字货币支付系统

后端 API 服务

● 服务运行中

API 端点

  • POST /api/wallet/derive 从助记词或私钥派生地址
  • POST /api/wallet/sign-message 签名消息
  • POST /api/wallet/sign-transaction 签名交易
  • POST /api/wallet/save 保存加密钱包
  • POST /api/wallet/get 获取钱包（解密到内存）
  • GET /api/wallet/list 列出所有钱包
  • POST /api/wallet/delete 删除钱包

API 文档

  • GET /api-docs/ui Swagger UI - 交互式 API 文档
  • GET /api-docs OpenAPI 规范 (YAML)

服务信息：

• 端口: 6543

• Private 目录: ./private

• 加密算法: Argon2id + AES-256

• 所有私钥和助记词均经过加密存储

• API 文档标准: OpenAPI 3.0.3
